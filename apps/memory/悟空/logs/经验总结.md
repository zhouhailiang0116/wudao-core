## 三、核心概念

### 1. 强化学习三要素
```
状态（State）→ 动作（Action）→ 奖励（Reward）
```

### 2. Q-Learning
```python
Q[(state, action)] = Q[state,action] + α*(reward + γ*maxQ(next_state) - Q[state,action])
```

### 3. 探索vs利用
```python
epsilon = 0.3  # 30%概率随机探索
epsilon *= 0.98  # 每回合衰减
```

### 4. 奖励函数设计
- 到达目标：+100~+200
- 碰撞/超时：-50~-80
- 存活：+0.5/步
- 距离奖励：-dist/15

## 五、关键教训

1. **强化学习需要试错** - v3撞墙274次才学会绕路
2. **状态设计要平衡** - 太细状态爆炸，太粗学不到
3. **奖励函数决定方向** - 设计不好会学歪
4. **epsilon衰减** - 从探索到利用的平滑过渡
5. **多智能体** - 对抗比协作更容易学习
